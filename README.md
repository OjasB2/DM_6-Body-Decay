# DM_6-Body-Decay
This repository contains the Python code used for the numerical analysis in my master's thesis, *Explaining the KM3NeT Ultra-High-Energy Neutrino Event with Heavy Decaying Dark Matter*.

The code computes prompt spectra for the six-body decay of DM candidate X to neutrinos via SU(2) triplet mediator $\Delta$:

```math
\chi \to 3\Delta \to 6\nu
```

using fragmentation functions from the [HDMSpectra](https://github.com/nickrodd/HDMSpectra) package. It then computes Galactic and extragalactic neutrino and photon fluxes, best-fit parameters for the [KM3NeT](https://www.nature.com/articles/s41586-024-08543-1) event, lifetime constraints from the observations of the [LHAASO](https://arxiv.org/abs/2210.15989) observatory, and the available model parameter-space.
